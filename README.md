# 蔡叔叔智慧排班 v0.5

## 網址
- 員工前台：`index.html`
- 主管後台：`admin26.html`

## 本次調整
- 員工首頁移除班次數與計薪工時統計。
- 員工首頁只保留「下一班」資訊。
- 主管後台新增「可排時間設定」。
- 主管可設定：
  - 開放填寫起日與迄日
  - 員工可填寫的排班日期起日與迄日
  - 區段名稱、提示文字、啟用／停用
- 員工只有在開放期間內才能新增或修改。
- 員工可排時間改為月曆總覽。
- 月曆會顯示：
  - 已填可排時間
  - 整天不可排班
  - 尚未填寫
  - 不在本次開放日期區段的日期
- 點選日期後，可輸入精確至 30 分鐘的最早上班及最晚下班時間。

## 示範設定
7/17～7/20 開放填寫 8/1～8/15 的可排時間。

## 注意
目前仍使用 localStorage，因此必須在同一瀏覽器才會共用資料。正式跨裝置使用前仍需串接 Firebase。


v0.4.1:
- 新增店家設定頁
- 員工類型調整為：正職/工讀/外籍學生/其他
- 外籍學生預設20小時(規格)
- 新增可排時間Dashboard預留頁

v0.4.2:
- 店家設定正式生效：營業起訖時間、時間間隔、每週公休日（可多選）、休息時段（14:30～16:00）。排班的時間選單依營業時間與間隔產生。
- 休息規則重新設計：改為「店家設定休息時段 + 每個工作是否套用休息」。班次涵蓋休息時段且該工作套用休息時，才自動扣除重疊分鐘；不套用休息的工作（如餅皮、備料）全額計薪。班次不再手動輸入休息分鐘，改為即時自動試算。
- 員工管理：員工類型改為 正職/工讀/外籍學生/其他；選外籍學生自動帶入預設 20 小時（可修改）；新增「主要工作（平日／假日）」欄位。
- 工作管理：新增「是否套用休息」開關；保留前置作業星期與提早分鐘。
- 排班：可排員工依「主要負責 → 可做工作 → 剩餘工時 → 可排時間」排序，主要負責者以 ★ 標示，並顯示不可排原因。
- 新增「工時總覽」：員工 × 一週七天矩陣，顯示每日與每週合計、對照上限；接近上限標黃、超過標紅；外籍學生標示且僅提醒不阻擋。可切換週次（草稿與已發布班次皆即時試算）。
- 新增「可排時間總覽」：月曆／日期／員工三種模式，Dashboard 顯示已填／未填人數與未填名單。

v0.5（進行中）:
- 排班改為直式時間軸，支援日檢視（橫軸工作）與週檢視（橫軸七天、顏色分工作），重疊班次自動並排。
- 員工端：我的班表分即將到來／已結束（含備註與休息時段）；填寫可上班時間支援單日快速✓✗與整週快速套用（可設定固定時段）；非開放期間可唯讀檢視近三個月紀錄。
- 新增「每日需求模板」：設定每星期各工作需要幾人，排班頁一鍵「套用需求」自動建立班次並依主要負責/工時指派最適人選，找不到人則標為待指派。
- 總覽新增缺人警訊（未來 14 天依需求模板）與未指派班次提醒。
