# 蔡叔叔智慧排班 v0.2

## 網址分流
- `index.html`：員工前台，只輸入員工編號。
- `admin26.html`：主管後台，員工管理、工作管理與排班。

部署到 GitHub Pages 後：
- 員工網址：`https://帳號.github.io/repository/`
- 主管網址：`https://帳號.github.io/repository/admin26.html`

## 本次調整
- 員工前台與主管後台完全拆成不同 HTML。
- 排班新增／編輯流程改為先選日期、工作、開始與結束時間，最後才選員工。
- 員工下拉選單自動分成「可排班員工」與「不可排班／需確認」。
- 判斷依據包含：
  - 員工是否會做該工作
  - 是否填寫當日可排時間
  - 班次是否超出可排時間
  - 是否與既有班次重疊
  - 排入後是否超過每週工時上限
- 第一版仍允許主管選擇不符合條件的員工，以保留臨時協調的彈性。

## 注意
目前仍使用同一瀏覽器的 localStorage。員工與主管若使用不同裝置，資料不會同步；下一階段需串接 Firebase。
