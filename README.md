# 蔡叔叔智慧排班 v0.3

## 網址
- 員工前台：`index.html`
- 主管後台：`admin26.html`

## 本次調整
- 員工首頁移除班次數與計薪工時統計。
- 員工首頁只保留「下一班」資訊。
- 主管後台新增「可排時間設定」。
- 主管可設定：
  - 開放填寫起日與迄日
  - 員工可填寫的排班日期起日與迄日
  - 區段名稱、提示文字、啟用／停用
- 員工只有在開放期間內才能新增或修改。
- 員工可排時間改為月曆總覽。
- 月曆會顯示：
  - 已填可排時間
  - 整天不可排班
  - 尚未填寫
  - 不在本次開放日期區段的日期
- 點選日期後，可輸入精確至 30 分鐘的最早上班及最晚下班時間。

## 示範設定
7/17～7/20 開放填寫 8/1～8/15 的可排時間。

## 注意
目前仍使用 localStorage，因此必須在同一瀏覽器才會共用資料。正式跨裝置使用前仍需串接 Firebase。
