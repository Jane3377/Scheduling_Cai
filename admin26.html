特定休息日（臨時公休）

指定單一日期整店休息（例如臨時公休、內部訓練），該日在排班與員工填寫都會鎖起來反灰。每週固定的公休請用上方店家設定的「每週公休日」；國定假日請用下方管理。

國定假日

匯入後只會在月曆上「標示」假日名稱，不會自動公休；是否公休由您在每個假日按「設為公休」決定。也可用下方欄位自行新增假日（新年度或補班補假調整時使用）。

批次貼上匯入（自行匯入其他年度，不用等更新）

每行一個假日，格式「日期 名稱」，例如 2028-01-01 元旦。可從政府「辦公日曆表」或 Excel 直接複製貼上，系統會自動辨識日期。

後台 PIN 碼

進入後台管理時需輸入的密碼，預設 1234。修改後下次進入即生效。

資料備份與還原

用於換裝置、搬站或定期備份。建議平時就用「完整備份」匯出一份保存。

完整備份（推薦） 一個檔包含所有資料（員工、工作、班次、可上班時間、固定班次與設定）。
分項備份 只匯出／匯入單一類別，適合把某類資料搬到另一間店。匯入會覆蓋該類別現有資料。

資料清理

清除舊的排班班次與可上班時間，讓資料量不會無限增加。工時是依班次即時計算，清掉舊班次即清掉舊工時。員工、工作、固定班次設定不會被清。此動作無法復原，建議先用上方「完整備份」匯出。

自動清除 開啟系統時，會自動清除「超過下列天數」的舊排班與可上班時間。預設 365 天（一年）；填 0 ＝ 不自動清除。
立即清除 馬上清除 180 天前的舊排班與可上班時間（不影響員工、工作、固定班次）。

清空所有資料

清空所有資料（員工、工作、班次、可上班時間等）。此動作無法復原，建議先用上方「完整備份」匯出保存。

使用說明 / 快速上手

點各段標題可展開。忘記怎麼操作時可隨時回這裡查。

🚀 第一次設定（只做一次）
  1. 設定與維護 → 店家設定：填店名、營業時間、休息扣薪時段。
  2. 工作項目：建立工作（內場、外場…），設顏色、是否平日/假日限定、要不要扣休息。
  3. 員工管理：新增員工——編號、身份（外籍學生會提醒工時上限）、時薪、可做的工作、主要負責的平日/假日工作、勞健保備註。
  4. 設定與維護 → 國定假日：一鍵匯入台灣國定假日；特定休息日設定公休。
  5. 排班管理 → 固定班次設定：把「每週固定的樣子」建好（週一～週日與國定假日各自的班），之後可一鍵套用。
📅 每週怎麼排班
  1. 可上班時間 → 開放設定：按「開放填下週」或「開放填下個月」讓員工填。
  2. 員工填完後回 排班管理：按「套用固定班次」自動照模板排好、能排的人自動補上（再按一次只補空缺、不會重複）。
  3. 要微調：在空白時段上下拖曳框選時段新增班次；班次色塊右上角點「⇄」可直接換人。想沿用上週按「複製上週班表」。
  4. 排好後按「公布本週」，員工才看得到（公布前是灰色斜紋草稿）。需要紙本按「列印週班表」。

💡 總覽會提醒「有班次沒排人 / 沒公布」，點提醒可直接跳到那一天。

🤖 套用固定班次：自動帶人怎麼挑

按「套用固定班次」時，系統會照當天的固定班次，自動把最適合的人帶進去；找不到合適的人就先留「待指派」。挑人分兩關 —— 先看「能不能排」，再看「誰優先」。

第一關 · 能不能排（要全部符合才會被自動帶入）

  • 會做這個工作
  • 有填可上班時間，且當天不是「不可上班」
  • 班次時間落在他可上班的時段內
  • 當天沒有其他時間重疊的班次
  • 排進去不會超過每週工時上限

☝ 有任何一項不符，這個人這次就不會被自動帶入。

第二關 · 誰優先（符合的人裡，優先度高的先排）

條件優先度
當天的「主要負責人」（平日看主要平日、假日看主要假日）★★★ 最高
會做這個工作★★ 高
本週剩餘工時較多（用來平衡大家工時）★ 中
有填可上班時間· 低

重點

  • 設為「主要負責」的人，只要符合資格，幾乎一定被排第一 —— 這就是「自動帶入負責人」。
  • 同一時段有多位主要負責人時，本週排得比較少的先排，自動平衡工時。
  • 主要負責人若沒填可上班／當天不可／時段不符／會超時，就不會被帶入，可能留白或由次順位頂上。
  • 補不滿會提示「仍有 N 個待指派」；等員工更新可上班時間後再按一次就自動補上，不會重複。想手動改，點班次右上角「⇄」直接換人。
🔄 開放員工填可上班時間
  • 用「開放填下週 / 下個月」快捷，日期會自動帶好，確認即可儲存。
  • 勾「預設全部員工整天可上班」→ 只需標記少數請假的人，其餘視為可上班。
  • 檢視與代填：主管也能用日檢視／員工檢視，直接幫員工快速勾選。
⏱ 月底對帳與下載報表
  • 工時統計：可選週或月、依工時排序、篩選身份、搜尋員工，點某人展開明細。
  • 拿打卡表核對：填實際上下班、打勾確認（打卡對照）。
  • 看人力成本（時薪×工時），按「下載 CSV」給會計。
📱 員工怎麼用（可截圖給員工）
  • 打開員工班表系統，輸入員工編號登入。
  • 我的班表：看已公布班表、一鍵「加入行事曆」（iPhone／Google）、班表更新有橫幅通知。
  • 填寫可上班時間：月曆勾選、可「快速設定整週」、標記「整天不可上班」。
💾 備份與資料清理
  • 平時定期用「完整備份」匯出一份保存；換裝置或搬站可「還原完整備份」。
  • 「立即清除 180 天前資料」可手動清舊資料；超過 365 天會自動清（可調）。
  • 清理只影響舊班次與可上班時間，員工、工作、固定班次不會被清。
❓ 常見問題
  • 草稿與公布差在哪？灰色斜紋＝草稿，只有主管看得到；按「公布本週」後員工才看得到。
  • 假日怎麼算？週六、日與國定假日算假日，套用假日的工作與固定班次。
  • 外籍學生工時？系統會依每週上限提醒超時，但仍可排（保留彈性）。
  • 套用固定班次會重複嗎？不會。再按只補未指派的空缺，不會多加班次。
  • 右上角狀態燈？●已同步、◌儲存中、◍本機模式——確認資料有沒有存上雲端。

固定班次設定

設定每個星期各工作固定需要幾人；回排班管理按「套用固定班次」即可自動建立班次。
